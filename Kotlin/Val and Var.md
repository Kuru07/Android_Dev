_Val_ is used for the value. It is fixed.
If we want to change the value in between using _Var_
Types:
*Primitive data types*:
1) Integer
2) Float
3) Double
4) Boolean
5) Strings

```kotlin
  
fun main()  
{  
    var a=5  
    val x = 5  
    val y:Int = 6  
    val z:Float = 10.456F  
    val d = 5.67 //Double  
    val bool = true  
    val string = "Hello World"  
    a=22  
    println(a)  
    println(x)  
    println(y)  
    println(z)  
    println(d)  
    println(bool)  
    println(string)  
}
```
